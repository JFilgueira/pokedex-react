# Pokedéx with React.js

##[Live preview on Vercel](https://pokedex-react-flax.vercel.app/)

## Project's Scope
* List Pokémons basic info in cards
* View detailed info of each Pokémon
* Favorite Pokémons
* Search Pokémon by it's Name
* Dark mode
* Responsive design

![]()

## Getting started

### Running 
```
npm install
npm start
```
